# Getting Started with OpenAI ChatGPT

### Reference Documentation
Call OpenAI ChatGPT APIs in Spring Boot

### Dependency
```bash
Java 21    
Spring Boot: 3.2.3
Spring OpenAI: 0.8.0
Lombok : 1.18.30

```
#### Endpoint 
http://localhost:8080/chat [GET]


#### Example request:
http://localhost:8080/chat?prompt=Make%your%question!
